Overview

Created by John Horton Conway in 1970, Conway's game of Life is a two-dimentional cellular automaton where each cell has two states: Alive or Dead. The game istelf exists in an infinite Two-Dimentional grid of square cells.

Although it is a 'game' it is one that requires no player. This means that the outcome of the game is solely reliant on the starting conditions. That being which cells are which state when the the atomaton is started.

Rules

Each time step every cell interacts with its 8 neighbours and decides which state to change to depending on the states of the surrounding neighbours. At each time increment, there are four events:

  1. Any living cell with less than two living neighbours dies
  2. Any living cell with two or three living neighbours lives
  3. Any livin cell with more than 3 neighbours dies
  4. Any dead cell with three neigbours becomes alive
However each cell should be checked simultaniously, meaning that if at time step 4 a cell dies, all other cells will still see it as alive until time step 5.

Patterns

Many different patterns have been observed in Conway's Game of Life, some of which are static, and once reached do not change without ouside interference (Called Still Lifes). Others oscillate between multiple states (Oscillators) and some move across the grid indefinately (Gliders).

As well as these types there are 'Guns' which are patterns that oscillate and in their oscillation create gliders.
